### Installation

* step 1

    `$ python manage.py makemigrations`
* step 2

    `$ python manage.py createsuperuser`
* step 3

    `$ python manage.py makemigrations app`
* step 4

    `$ python manage.py runserver 8090`
* step 5

    open http://127.0.0.1:8090/admin/
